# 미적분·최적화, 눈으로 보기 (Calculus & Optimization, Visualized)

기울기를 따라 가장 낮은 곳으로 — 딥러닝·최적화의 수학 토대를 **수식 없이 직접 만지며** 익히는 인터랙티브 학습 자료입니다. 모든 곡선·기울기·넓이·경로가 브라우저에서 실시간으로 계산됩니다.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

## 👋 처음이세요? — 0번 입문 챕터

수식 없이 "무엇을·왜 배우는지" 먼저 둘러보는 5분 직관 입문입니다 → **[`00-intro.html`](00-intro.html)** (목차의 "처음이세요?" 배너에서도 진입)

| Step | 내용 |
|------|------|
| 00 · 어디에나 | 딥러닝·회귀·넓이/확률·경제·공학 속 미분과 최적화 |
| 01 · 미분 = 기울기 | 곡선 위 점을 끌어 그 점의 기울기(접선) 보기 |
| 02 · 최적화 = 가장 낮은 곳 | 공을 굴려 골짜기로 — 보폭(학습률)의 효과 |
| 03 · 코스 지도 | 16개 챕터 흐름 + 추천 순서 |

## 시작하기

빌드·의존성 없음. `index.html` 을 브라우저로 열면 끝입니다.

```bash
open index.html                 # macOS
python3 -m http.server 8000     # → http://localhost:8000
```

## 챕터 (16개)

**PART 1 · 미분 직관 (1변수)** — 01 함수와 변화 · 02 극한·순간변화율 · 03 도함수
**PART 2 · 미분 도구** — 04 연쇄법칙(→역전파) · 05 2차도함수·볼록/오목
**PART 3 · 다변수와 그래디언트** — 06 다변수·등고선 · 07 편미분 · 08 그래디언트 · 09 방향도함수
**PART 4 · 최적화 엔진** — 10 경사하강 · 11 학습률·모멘텀·지역최소 · 12 볼록성 · 13 뉴턴법
**PART 5 · 쌓기와 제약** — 14 적분·리만합 · 15 미적분 기본정리 · 16 제약 최적화(→정규화)

> 척추는 **도함수(03) → 연쇄법칙(04) → 그래디언트(08) → 경사하강(10)**. 처음이라면 0번부터 순서대로 권합니다.

## 구조

```
index.html          목차 허브
shared/theme.css    공통 디자인 시스템 (다크 테마)
shared/lib.js       공통 헬퍼 + 미적분 엔진(VZ.CA: 좌표계·함수곡선·미분/적분·등고선·경사하강) + 차트(VZ.linePlot)
NN-주제.html         챕터별 단일 파일 (각자 독립 실행)
```

## 설계 원칙

- **실시간 계산**: 미분·적분·그래디언트·경사하강이 모두 실제 수치 로직으로 동작 (하드코딩 결과 없음)
- **수식 없는 직관 우선**: 기호를 외우는 대신 슬라이더로 만지며 눈으로 — 기호는 등장할 때마다 풀이
- **단계별 분해**: 각 챕터 4스텝, 한 줄 직관 → 인터랙션 → 결론, 그리고 머신러닝·실무로 잇는 마무리
